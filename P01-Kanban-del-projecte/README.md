# P01: Kanban del projecte

## Breu descripció

Com en projectes anteriors, la metodologia **Kanban** és vital pel bon funcionament del mateix.  
Per aquest motiu, cal que a partir del **product backlog** que se us proporcioni feu el seguiment de les vostres realitzacions del projecte.

Cada setmana es farà una revisió de l’estat del **Planner**, per això és molt important que no el deixeu per darrera hora.

Aquí teniu el **product backlog** d’aquest projecte.
