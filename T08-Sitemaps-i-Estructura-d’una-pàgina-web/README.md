# T08: Sitemaps i Estructura d’una pàgina web

## Breu descripció

A **EverPia**, la consultora creix i els projectes web cada vegada són més complexos.  
Els clients ja no volen només “tenir una web”, volen que **funcioni**, que sigui **clara, intuïtiva i professional**.

Per això, abans de continuar amb el projecte **P03: Rèplica web**, el departament de desenvolupament organitza una **formació interna** per assegurar que tots els equips tinguin una base sòlida sobre **com s’estructura una web professional** i **com es defineix el mapa de navegació (sitemap)**.

Aquesta sessió està dissenyada com una **màsterclass pràctica** amb exemples reals i exercicis curts.  
Servirà per entendre la lògica darrere d’una web, més enllà de l’estètica: **com s’hi organitza la informació, com s’hi pensa la navegació i com es connecten les pàgines entre si**.

---

## Objectius de la sessió

- Comprendre les parts essencials d’una pàgina web (**capçalera**, **cos**, **peu de pàgina**).
- Distingir entre **pàgines estàtiques** i **entrades dinàmiques**.
- Entendre què és un **sitemap** i per a què serveix (tant a nivell tècnic com visual).
- Saber **com representar la jerarquia** d’un lloc web.
- Crear un **document tècnic explicatiu en format Markdown**, que servirà com a referència per a futurs projectes.
