# Activitat: Relació entre ODS i ASG

## Breu descripció

Mitjançant unes **activitats concretes**, treballarem els **ASG (Ambiental, Social i de Governança)** i la seva relació amb els **Objectius de Desenvolupament Sostenible (ODS)**.  
L’objectiu és entendre com les empreses poden aplicar aquests conceptes per millorar el seu impacte sostenible i social.

---

## Objectius específics de la tasca / Finalitat de la tasca

- **Recordar** els Objectius de Desenvolupament Sostenible (**ODS**).  
- **Treballar** els conceptes **ASG**.  
- **Relacionar** els **ODS** amb els **ASG**.  
- **Resoldre qüestions i exemples reals** d’empreses per comprendre com apliquen aquests principis.
